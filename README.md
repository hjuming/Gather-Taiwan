## Content Boundary

本網站目前為策略概念與招商溝通用途。

- 不構成正式活動公告
- 不提供票務資訊
- 不列合作夥伴 logo
- 不把潛在接觸對象寫成合作事實
- 不使用活動型 JSON-LD
- 12 個 Gathering Theme / Calendar 皆為未來策展方向或 concept module，不代表即將舉辦
