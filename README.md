## 關於這個活動

**新熱炒運動**是一個翻轉台灣傳統熱炒文化的文化行動計畫。

塑膠紅椅、磨石子地板、圓形桌、生鐵鍋、霓虹招牌——台灣人只要看到這幾個元素，就知道這是屬於自己的場景。**熱炒，是台灣最日常的社交儀式。** 新熱炒運動翻轉這個空間，注入年輕世代的音樂與潮流元素，讓它成為音樂現場、潮流地標、文化出口。

## 活動亮點

### 🎵 台灣夢音樂祭（旗艦館）

進駐大稻埕碼頭貨櫃市集，打造結合霓虹燈管、磨石子地貼與塑膠紅椅的沈浸式音樂現場，提供台灣味限定特調飲品。

**演出陣容 Lineup**

**壓軸 Headliners**
- 夾子電動大樂隊 The Clippers Band — 綜藝搖滾・電子花車・那卡西
- 閃亮三姐妹 Shining Sisters — 台客電音・電子花車家族

**主秀 Main Acts**
- 美秀集團 Mxsh Group — 賽博台客・歌廳秀電搖
- 茄子蛋 EggPlantEgg — 台語情歌・浪子回頭
- 落日飛車 Sunset Rollercoaster — City Pop・夜市浪漫國際

**暖場 Opening Acts**
阿跨面 Akuma・百合花 Lily Flower・李竺芯・163braces・someshiit 山姆・王ADEN・陳泳希 JUD・派偉俊 Patrick Brasca

### 🚌 流動熱炒車（Mobile Rechao）

改裝復古電子花車從台北出發，巡迴六都商圈。車上設置鐵窗花拍貼機、限定熱炒菜單、不定期街頭快閃演出。

### 📸 社群挑戰 #TaiwanSensibility

拍下你身邊的老公寓、夜市角落、磨石子地板、鐵窗花紋路，標記 `#TaiwanSensibility` 與 `#新熱炒運動`，有機會獲得活動早鳥票。

## 關於這個網站

本站為 **Neo-Rechao 新熱炒運動**的官方活動網站，採用純靜態 HTML 架構，部署於 Cloudflare Pages。

### 技術規格

- **架構**：純靜態 HTML + Vanilla CSS + JavaScript
- **字型**：Noto Serif TC・Noto Sans TC（Google Fonts）
- **部署**：Cloudflare Pages
- **SEO**：Open Graph・Twitter Card・JSON-LD MusicEvent 結構化資料・XML Sitemap
- **響應式**：支援手機（≤480px）、iPad（≤768px）、桌機

### 專案結構

```

├── index.html          # 主網站
├── support.js          # 執行環境
├── _headers            # Cloudflare 安全 headers
├── _redirects          # Cloudflare 重定向規則

└── uploads/            # 圖片素材
    └── neo-rechao-og.png  # 社交分享預覽圖
```
